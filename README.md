# My Page

A personal link page built with [Next.js](https://nextjs.org) (App Router) and Tailwind CSS.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the page.

Edit `app/page.tsx` to add your name, bio, and links.

## Deploy

Deploy for free on [Vercel](https://vercel.com) or any platform that supports Next.js.
